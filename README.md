# robrowan.docker

Deploys docker-ce

## Getting Started

### Prerequisites

* Ansible

### Installing

```
$ ansible-galaxy install robrowan.docker
```

## Use

### Example playbook

```
- hosts: all
  roles:
    - robrowan.docker
```
